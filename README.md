# FH Nesjs Course
## 04- Teslo Shop API

1. Clonar proyecto
2. ```npm i```
3. Clonar ``` .env.template ``` y renombrar a ``` .env ```
4. Cambiar variables de entorno
5. Levantar base de datos:
```
docker-compose up -d
```

6. Ejecutar en modo dev:
``` npm run start:dev ```
